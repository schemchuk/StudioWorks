Задача: Система Управления Библиотекой
Цель:
Разработать систему управления библиотекой, используя принципы объектно-ориентированного программирования (ООП), такие как инкапсуляция, наследование, полиморфизм. Has-a  композиция.

Задача:
Вам необходимо спроектировать и реализовать систему для библиотеки, которая позволяет управлять книгами и пользователями библиотеки. Система должна обладать следующими возможностями:

Добавление и удаление книг в каталог.
Регистрация и удаление пользователей библиотеки.
Поиск книг по различным критериям (например, по автору, названию или жанру).
Возможность для пользователей брать книги на дом и возвращать их.
Требования к системе:
Класс Book:

Атрибуты: название, автор, год издания, жанр, статус (доступна для выдачи или нет).
Методы: конструктор, геттеры и сеттеры для атрибутов.
Класс User:

Атрибуты: имя, идентификатор (ID), список взятых книг.
Методы: конструктор, методы для взятия книги на дом и возвращения книги, геттеры и сеттеры для атрибутов.
Класс Library:

Атрибуты: список книг в библиотеке, список зарегистрированных пользователей.
Методы: добавление и удаление книг, регистрация и удаление пользователей, поиск книг, выдача книг пользователю, прием возвращенных книг.
Рекомендации:
Используйте наследование для создания специализированных типов книг или пользователей, если это необходимо.
Примените полиморфизм, например, для реализации различных стратегий поиска книг.
Рассмотрите возможность использования композиции, например, включив список книг как часть класса User для учета взятых книг.
Задание:
Реализуйте указанные классы с использованием Java.
Создайте тестовый пример, который демонстрирует работу вашей системы: регистрация пользователей, добавление книг в каталог, выдача книги пользователю и её возврат.

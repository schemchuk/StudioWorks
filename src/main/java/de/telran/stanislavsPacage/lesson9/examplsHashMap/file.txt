
put(Object(key), Object(value)) - метод используется для связывания указанного значения с указанным ключом
get(Object(key)) - метод используется для получения значения по указанным ключом
size()	- метод используется для возврата количества пар ключ/значение
remove(Object) - метод используется для удаления
isEmpty() - метод используется для проверки наличия пар ключ-значение. Если сопоставления не существует, возвращается true
clear()             - метод используется для очистки и удаления всех элементов

entrySet() - метод используется для создания набора из тех же элементов

keySet() - метод используется для возврата представления Set ключей

containsKey(Object) - метод используется для проверки того, содержится ли конкретный ключ или нет
containsValue(Object)	- метод используется для проверки того, отображается ли конкретное значение одним или несколькими ключами, возвращает True, если это значение сопоставляется с любым ключом
